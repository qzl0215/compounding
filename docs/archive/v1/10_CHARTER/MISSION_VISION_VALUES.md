# Mission Vision Values

## Mission

用一套可计算、可继承、可审计的虚拟公司治理操作系统，让 AI 项目从第一天起具备长期复利能力。

## Vision

让任何新项目都能在统一治理与知识底座之上，以最小熵增持续进化。

## Values

- 单一真相入口
- 先证据后判断
- 先收敛后扩张
- 复杂度净减少
- 可回滚与可审计
