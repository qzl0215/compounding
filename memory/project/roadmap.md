# 路线图

## 当前阶段

Structural Entropy Reduction（持续收口）

## 当前里程碑

下一轮高 ROI 结构收口候选评估

## 里程碑成功标准

- 明确下一条单一结构主线，只选一个会持续制造熵增的边界推进
- 不新增新状态源、重型框架或平行读模型
- 本地 production 继续稳定，且保持无 `pending dev`

## 当前优先级

继续做高 ROI 结构 review，优先评估 portal 读模型聚合点与 release 兼容壳。

## 下一阶段方向

- 若进入 portal 方向，优先把首页/任务页/发布页共享的聚合逻辑再切薄一层
- 若进入 release 方向，优先把历史兼容壳继续下沉，避免反向污染主读链
- 继续在不增加新状态源的前提下退休低价值解释层和重复外壳
