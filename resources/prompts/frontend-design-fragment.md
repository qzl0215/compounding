## Frontend Design Directive

When generating ANY frontend code (components, pages, applications, styles):

### MANDATORY: Design Thinking First

Before writing any code, establish:

1. **Aesthetic Direction** - Pick ONE and commit:
   - Brutalist / Editorial / Dark OLED Luxury / Retro-Futuristic
   - Organic/Natural / Minimalist / Maximalist / Playful

2. **Typography** - Distinctive pairing required:
   - NO Inter, Roboto, Arial, system fonts
   - Use Google Fonts with personality

3. **Color System** - Use CSS variables:
   - Bold choices with sharp accents
   - Create atmosphere, not flatness

4. **One Unforgettable Element** - What makes this design memorable?

### FORBIDDEN

```
🚫 Inter, Roboto, Arial, system-ui as primary font
🚫 Purple gradients on white backgrounds
🚫 Rounded cards with centered grid layout
🚫 Space Grotesk (overused by AI)
🚫 Flat solid-color backgrounds
🚫 Scattered micro-interactions
🚫 Generic "AI slop" aesthetics
```

### Implementation Matching

```
Minimalist design → Maximum restraint, precision
Maximalist design → Bold colors, dense layouts
Luxury design → Dark backgrounds, gold accents, thin serifs
```
